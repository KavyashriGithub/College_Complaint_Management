College Complaint Management System (CCMS)
📌 Project Overview

The College Complaint Management System (CCMS) is a web-based application developed to simplify and digitalize the complaint handling process within a college campus.
Students can raise complaints related to academics, hostel, transport, facilities, or other issues, while administrators can manage, track, and resolve them efficiently.

This project is built using the MERN Stack:

MongoDB – Database
Express.js – Backend Framework
React.js – Frontend Library
Node.js – Runtime Environment
🚀 Features
👨‍🎓 Student Module
Student Registration & Login
Raise New Complaints
View Complaint Status
Track Complaint History
Dashboard for complaint management
👨‍💼 Admin Module
Secure Admin Login
View All Complaints
Update Complaint Status
Resolve/Delete Complaints
Manage Students and Complaint Categories
🔐 Authentication & Security
JWT Authentication
Protected Routes
Password Encryption
📊 Additional Features
Responsive UI
Real-Time Status Updates
Search & Filter Complaints
Role-Based Access

⚙️ Installation & Setup
1️⃣Navigate to Project Folder
cd College_Complaint_Management_System
2️⃣Install Frontend Dependencies
cd client
npm install
3️⃣Install Backend Dependencies
cd ../server
npm install

🔑 Environment Variables
Create a .env file inside the server folder and add:
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

▶️ Run the Project
Start Backend Server
cd server
npm start
Start Frontend
cd client
npm start

🌐 Application Flow
1.Student registers/logs in
2.Student submits a complaint
3.Complaint stored in MongoDB
4.Admin views complaints
5.Admin updates complaint status
6.Student tracks resolution progress

🎯 Objectives
1.Reduce manual complaint handling
2.Improve communication between students and administration
3.Maintain transparency in complaint resolution
4.Provide faster and organized complaint tracking
